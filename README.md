# Calculator

A simple Java-based calculator application designed to compute numbers efficiently. Great for basic arithmetic operations.

---

## Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- ✅ Clean and simple interface  
- ⚙️ Built with core Java (no external libraries)
